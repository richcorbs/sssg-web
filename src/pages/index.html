a simple website framework

This is a lightweight, opinionated framework for developers who want simplicity, speed, and ease in creating, maintaining, and deploying static websites.

How do I get started?

  1. 1. Download the binary from here. This is the only dependency.

  2. 2. Run sssg --init to scaffold your new site in the current directory.

  3. 3. Run sssg --dev to start the dev server. It will tell you where your site is running.

  4. 4. Modify your content in the ./src directory and your changes will be reflected in the browser immediately.

  5. 5. Run: sssg --register <domain>

    where <domain> is the domain you want to deploy your site to.

    The domain you choose can either be your own custom domain or a subdomain on sssg.dev like my-site.sssg.dev. If it is your own domain you'll need to create a CNAME record pointing to www.sssg.dev.

    When you register a domain we will also create a random staging domain for you.

  6. 6. To deploy run:

    sssg --deploy --staging

    or

    sssg --deploy --production

    Deploying to staging will not affect your production site.

    Go here to learn more about deploying your site.

    * If you do not want to deploy to our servers, run sssg --build and then manually deploy the ./dist directory to your server.

Tips:

  • Edit the src/pages/index.html file to change this page. The page should hot reload automatically after you save.
  • Edit the src/layouts/Default.html file to change the default layout.
  • Edit the src/assets/css/custom.css file to change the custom styles. We've made it easy for you to change the brand and accent colors.
  • Add a page in src/pages with just markdown. Make sure it has an extension of .md.
