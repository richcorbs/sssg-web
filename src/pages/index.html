a simple website framework

This is a lightweight, opinionated framework for developers who want simplicity, speed, and ease in creating, maintaining, and deploying static websites.

How do I get started?

1. Download the binary from here. This is the only dependency.

2. Run sssg --init to scaffold your new site in the current directory.

3. Run sssg --dev to start the dev server. It will tell you where your site is running.

4. Modify your content in the ./src directory and your changes will be reflected in the browser immediately.

5. Run sssg --register --domain <domain> to choose where your site will run. This will reserve <domain> for your site AND create a "staging" subdomain for you on sssg.dev.

6. Run:

sssg --deploy --staging

or

sssg --deploy --production

to deploy your site to our server. Deploying to staging will not affect your production site.

Tips:

  • Edit the src/pages/index.html file to change this page. The page should hot reload automatically after you save.
  • Edit the src/layouts/Default.html file to change the default layout.
  • Edit the src/assets/css/custom.css file to change the custom styles. We've made it easy for you to change the brand and accent colors.
  • Add a page in src/pages with just markdown. Make sure it has an extension of .md.
